# Day 04 – Web Basics & HTTP Observation

## Website Observed
I opened example.com and inspected the Network tab.

## One Request I Observed
Request URL: https://4k wallpaper.com/  
Request Method: GET  
Status Code: 200 OK  

This request loads the main webpage.

## One Status Code
200 OK means the request was successful.

## HTTP vs HTTPS
HTTP is not secure.  
HTTPS is secure and encrypted.

## Request and Response
Request is sent by browser.  
Response is sent by server.

## One Thing I Learned
One webpage loads many requests like images, CSS, and JavaScript files.
